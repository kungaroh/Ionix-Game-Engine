Ionix API Reference

All objects on ObjectLayer have a physics component.
This component can be controlled via properties in edior:

bool: TogglePhysicsOff

If True, stops the above physics component being added to the object.

bool: isTrigger

True or False to state if trigger or not.

bool: isStatic

Is the object static or not.

bool: isKinematic

Kinematic or not.

bool: LockRotation

If True, stops the physics component from rotating.

float: density

Range between 0 and 1.

float: friction

Range between 0 and 1.

float: restitution

Range between 0 and 1.

float: ColliderWidth

Float determines the width of the collider on the object.

float: ColliderHeight

Float determines the height of the collider on the object.

string: tag

Give the object a tag name (access with the get_object_tag hook)

string: ScriptComponent

Expects the name of a script, and will add this script for you.

You can have more than one script by adding more components.

string: SpriteComponent

Manually renders a .png from "TestProjectData/textures/" to the object.

set_object_position(int uid, float xPos, float yPos)

Sets the position of an object using its unique ID (uid) and the specified x and y coordinates.

get_object_position(int uid)

Returns the position of an object based on its unique ID (uid).

set_object_rotation(int uid, float rotation)

Sets the rotation of an object using its unique ID (uid) and the specified rotation angle.

get_object_rotation(int uid)

Returns the rotation of an object based on its unique ID (uid).

get_object_scale(int uid)

Gets the scale of the object

set_object_scale(int uid, float scaleX, float scaleY)

Sets the scale of the object

get_object_tag(int uid)

Gets the tag of the object

get_object_name(int uid)

Gets the name of the object

uid_from_name(string gameObjectName)

Returns the unique ID (uid) of an object using its name.

instantiate(string nameOfObjToSpawn, int xPos, int yPos)

Uses an existing object name to create a clone at a specified position.

Object names, for now, are defined in CPP (test classes) - See QAV11_TestGame.cpp

Alternatively, you can use change_gameObject_name to modify an existing object name via script, and use that new name instead.

disable_object(int uid)

Disables an object using its unique ID (uid).

is_enabled(int uid)

Checks whether an object is enable using the it's unique ID (uid).
Returns bool.

enable_object(int uid)

Enables an object using its unique ID (uid).

remove_object(int uid)

Removes an object from the scene using its unique ID (uid).

change_gameObject_name(int uid, string newName)

Changes the name of an object using its unique ID (uid).

get_mouse_x_position()

Returns the current X position of the mouse in screen coordinates.

Returns the current Y position of the mouse in screen coordinates.

is_mouse_down(Ionix_LeftClick)

Checks if the specified mouse button (e.g., Ionix_LeftClick) is currently pressed.

is_mouse_held(Ionix_RightClick)

Checks if the specified mouse button (e.g., Ionix_RightClick) is being held down.

is_mouse_up(Ionix_MiddleClick)

Checks if the specified mouse button (e.g., Ionix_MiddleClick) was released.

is_key_down(Ionix_Space)

is_key_held(Ionix_LeftShift)

is_key_up(Ionix_Return)

Checks if the specified key (e.g., Ionix_Return) was released.

RoundToInt(float number)

Rounds a number to the nearest integer. If the fractional part is 0.5 or greater, rounds up.

FloorToInt(float number)

Returns the largest integer less than or equal to the specified number.

CeilToInt(float number)

Returns the smallest integer greater than or equal to the specified number.

PowerOf(float base, float exponent)

Calculates the value of a base raised to a specified exponent.

AbsValue(float number)

Returns the absolute value of a number.

Rad2Deg(float radians)

Converts radians to degrees.

Deg2Rad(float degrees)

Converts degrees to radians.

Clamp(float value, float min, float max)

Restricts a value to be within a specified range.

Normalize(float x, float y)

Normalizes a 2D vector (x, y), returning its components as unit values.

Magnitude(float x, float y)

Calculates the magnitude (length) of a 2D vector (x, y).

RandomRangeInt(int min, int max [, int seed])

Returns a random integer between min and max. An optional seed parameter can be provided.

RandomRangeFloat(float min, float max [, int seed])

Returns a random floating-point number between min and max. An optional seed parameter can be provided.

PerlinNoise(float x, float y [, float cellSize])

Generates a Perlin noise value for the specified coordinates. An optional cellSize can be provided to scale the noise.

Interpolate(float t, float start, float end)

Linearly interpolates between start and end based on the parameter t.

BillinearInterpolate(float x, float y, float BL, float BR, float TL, float TR)

Performs bilinear interpolation based on the specified corner values and the parameters x and y.

newVector2(float x, float y)

Creates a new 2D vector with the specified x and y components.

vec1.add(vec2)

Adds the components of vec2 to vec1.

vec1.Sub(vec2)

Subtracts the components of vec2 from vec1.

vec1.Mult(float scalar)

Multiplies the components of vec1 by a scalar or by the components of another vector.

vec1.Div(float scalar)

Divides the components of vec1 by a scalar or by the components of another vector.

set_gravity_scale(int uid, float gravityScale)

Sets the gravity scale for an object specified by its unique identifier (uid).

add_force(int uid, float forceX, float forceY, float forcePosX, float forcePosY)

Adds a force to an object. The force is applied at a specific position (forcePosX, forcePosY).

add_box_collider(int uid, float posX, float posY, float width, float height, bool isTrigger)

Adds a box collider to an object. Specify the position, size, and whether it is a trigger.

add_circle_collider(int uid, float posX, float posY, float radius, bool isTrigger)

Adds a circle collider to an object. Specify the position, radius, and whether it is a trigger.

remove_collider(int uid, int colliderID)

Removes a collider from an object using its unique identifier (uid) and the collider ID.

get_velocity(int uid)

Gets velocity of the physics body attached to the object at (uid). Returns x, y components in world units (pixels/second)

set_velocity(int uid, float xvel, float yvel)

Sets the velocity of the physics body attached to the object at (uid) to (xvel, yvel) in world units (pixels/second)

raycast(int uid, int startX, int startY, int dirX, int dirY, float length)

Performs a raycast starting at (startX, startY) in a direction (dirX, dirY) for a specified length.

raycast_hit(int uid, int startX, int startY, int dirX, int dirY, float length)

Checks if the raycast intersects any objects in the specified direction and length.

first_raycast_hit(int uid, int startX, int startY, int dirX, int dirY, float length)

Finds the first object hit by the raycast along the specified direction and length.

box_cast(int uid, int x, int y, int x2, int y2)

Performs a box cast to detect collisions within a rectangular area defined by (x, y) to (x2, y2).

add_label(string text, int x, int y, int givenID, int groupID)

Adds a text label to the UI at the specified position (x, y). The label is assigned a unique identifier (givenID) and grouped under groupID.

add_button(string text, int x, int y, int sizeX, int sizeY, int givenID, int groupID)

Adds a button to the UI with the specified dimensions (sizeX, sizeY) and position (x, y). The button is assigned a unique identifier (givenID) and grouped under groupID.

is_button_pressed(int id)

Checks if a button with the specified identifier (id) has been pressed. Returns true if pressed, otherwise false.

set_group_position(int groupID, int x, int y)

Sets the position of a UI group identified by groupID to the new coordinates (x, y).

set_active(int elementID, bool state)

Sets the enabled state of a UI element identified by elementID to the new state (true/false) state).

update_ui_text(int elementID, string text)

Updates the text of a UI element identified by elementID to the new text text).

Ideal for use with score systems, where the UI changes frequently.

play_audio(string fileLocation, int loops)

Plays an audio file from the specified fileLocation. The loops parameter specifies how many times the audio should loop (-1 for infinite).

pause_audio(int handle)

Pauses the audio currently playing with the specified handle.

PauseAllAudio(int handle)

Pauses all audio currently playing.

resume_audio(int handle)

Resumes the paused audio associated with the specified handle.

fade_in(int ms, string fileLocation, int loops)

Plays an audio file with a fade-in effect over the specified duration in milliseconds (ms).

fade_out(int ms, int handle)

Fades out the audio associated with the specified handle over the specified duration in milliseconds (ms).

set_volume(int volume, int handle)

Sets the volume of the audio associated with the specified handle. The volume is typically between 0 and 128.

timed_play(int handle, float interval)

Plays the audio associated with the specified handle at a regular interval specified by interval.

random_timed_play(int handle, float min, float max)

Plays the audio associated with the specified handle at random intervals between min and max seconds.

ClearChannel(int handle)

Clears the audio channel with the specified handle.

ClearAllChannel()

Clears all audio channels.

get_window_dimensions()

Returns the width and height of the window.

get_camera_pos()

Returns the x and y coordinates of the camera. Note that a 'camera' is the coordinates of the top left corner of the screen in world coordinates, not the centre of the screen

set_camera_pos( x, y )

Sets the x and y coordinates of the camera. Note that a 'camera' is the coordinates of the top left corner of the screen in world coordinates, not the centre of the screen

set_texture( objUID, textureName )

Sets the texture on the sprite component (if any) atttached to the object at objUID. textureName is the name of the texture file.

set_animation_params( objUID, num_rows, num_frames, cur_row, speed_fps )

Sets an animation on the sprite component (if any) attached to the object at objUID. Animations are assumed to be made of square tiles in a texture, and a single animation mode must occupy a horizontal row in the texture. num_rows is the number of rows in the texture, cur_row is the row you want to play, num_frames is the number of tiles in the row to loop over, and speed_fps is the speed of the animation in frames per second

log(string input)

Writes the specified input to the console.

log_error(string input)

Writes the specified input to the console in red.

log_warning(string input)

Writes the specified input to the console in yellow.

CreateCoroutine( function_name, argument )

Creates a coroutine with the named function and argument. If the function is a member function of the object, the argument should be self. See the example.

YieldNextUpdate()

Called within a coroutine, this pauses execution until the next game loop update

YieldWaitForSeconds( seconds )

Called within a coroutine, this pauses execution for a time in seconds equal to the argument seconds.

call_function( script_name, function_name, uid )

This is used to call functions in other script components (on other objects, or even the same object). script_name is the name of the script, function_name is the name of the function to call, and uid is the uid of the object owning the script component. Note that this only supports calling procedures with no arguments (so, no inputs or outputs)
